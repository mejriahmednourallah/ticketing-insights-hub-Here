## 📌 Important Notes

1. **Supabase CLI Installation**
   - Script handles installation, but npm/bun installation doesn't work
   - Use: `curl ... | tar ...` or system package manager
   - See install-supabase-cli.sh for details

2. **Secrets Management**
   - All secrets externalized via .env
   - Never commit .env to git (already in .gitignore)
   - Local runtime env (.env.local.runtime) auto-generated by bootstrap

3. **Data Backward Compatibility**
   - loadTickets() tries Supabase first, falls back to CSV
   - Existing CSV bundle (public/data/issues.csv) still works
   - No breaking changes to Dashboard or SimilarityAnalysis UIs

4. **Cron Scheduling**
   - Uses PostgreSQL pg_cron extension (not external scheduler)
   - Runs every 5 minutes automatically
   - Can be triggered manually via bun run ingest:redmine:function
